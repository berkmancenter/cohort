size_for_text_area_tag(string, cols = 40)

Calculate a minimum size to display the entire contents of a body of text in a text_area_tag.

For example, given the following 

  str = "This is a test\n\nBacon!\n\nChunky Bacon."

when we call size_for_text_area(str,6) it would return <tt>"6x6". The first +6+ is because we said there were 6 columns. The second +6+ is because "Chunky" takes two rows, an empty space takes one, "Chunky Bacon." takes two, totalling four, plus two for good measure.
