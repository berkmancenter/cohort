send_individual_mail(contact_join)

Send a mail to an individual contact.

If the current delivery status does not match success then try sending the mail again.

If this raises an error catch the error and set it as the delivery status.

[Source]

    # File app/models/freemailer_campaign.rb, line 66
66:   def send_individual_mail(contact_join)
67:     begin
68:       if contact_join.delivery_status !~ /Success/
69:         Freemailer.deliver_from_template(self,contact_join.contact)
70:         contact_join.delivery_status = "Successful"
71:         contact_join.save
72:       end
73:     rescue Net::SMTPError => e
74:       puts "Delivery error rescued!"

76:       contact_join.delivery_status = e.to_s
